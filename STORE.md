## ストア掲載情報 (Store listing)

**名前**(manifestから自動): Board Image Downloader for Pinterest

**概要 (Summary)**(manifestのdescriptionが既定):
ボードの画像をオリジナル解像度で一括ダウンロード。ボード全体/直下のみ/セクション選択に対応し、ZIPまとめ保存も可能。

**詳細説明 (Description)**:

```
Pinterestのボードを開いてアイコンをクリックするだけで、ボード内の全画像を
オリジナル(最高)解像度で一括ダウンロードできます。

主な機能:
✔ ダウンロード範囲を選択 — ボード全体 / ボード直下のピンのみ /
   全セクションのみ / 特定のセクションだけ
✔ ZIPまとめ保存(既定) — 全画像を1つのZIPに固めて保存。
   フォルダ構造(ボード/セクション)もZIP内に再現されます
✔ 個別ファイル保存モードも選択可能
✔ オリジナル解像度(i.pinimg.com/originals)で取得。
   取得できないピンは自動的に736pxへフォールバック
✔ カルーセルピン(複数枚)も全スライド保存
✔ セクションページを開いた状態なら、そのセクションを自動選択
✔ 進捗バー表示・キャンセル対応

使い方:
1. Pinterestのボードページを開く
2. ツールバーのアイコンをクリック
3. 範囲を選んで「ダウンロード開始」

データ収集は一切行いません。すべての処理はブラウザ内で完結し、
外部サーバーへの送信はありません。

※ 本拡張機能はPinterest, Inc.とは無関係の非公式ツールです。
ダウンロードした画像は著作権法の範囲内(私的利用等)でご利用ください。
```

**カテゴリ**: ツール (Tools) ※「写真」でも可
**言語**: 日本語

**スクリーンショット**(最低1枚、1280x800 または 640x400):
ボードページでポップアップを開いた状態を撮影。
macOSなら Cmd+Shift+4 で範囲選択 → 1280x800にトリミング。
(撮ったらこのリポジトリに置いてくれれば加工します)

## プライバシー (Privacy practices)

**単一目的の説明 (Single purpose)**:

```
ユーザーが閲覧中のPinterestボードの画像を、ユーザーの明示的な操作により
ローカルに一括ダウンロードする。それ以外の機能は持たない。
```

**権限の使用理由 (Permission justifications)**:

- `downloads`:
  ```
  取得した画像(またはZIPアーカイブ)をユーザーのダウンロードフォルダに
  保存するために使用します。
  ```
- `offscreen`:
  ```
  Manifest V3のservice workerではBlob URLを生成できないため、
  画像をZIPアーカイブに変換しobject URLを生成する処理を
  offscreen documentで実行します。
  ```
- ホスト権限 `*.pinterest.com / *.pinterest.jp`:
  ```
  ユーザーが開いているボードのピン一覧をPinterestから取得するために
  使用します。ユーザーがポップアップから操作した時のみアクセスします。
  ```
- ホスト権限 `i.pinimg.com`:
  ```
  ピンの画像ファイル本体(Pinterestの画像CDN)をダウンロードするために
  使用します。
  ```

**リモートコード**: 使用していない (No, I am not using remote code)

**データ使用 (Data usage)**:
すべての項目で「収集しない」にチェック。
- 本拡張はユーザーデータを一切収集・送信・販売しない
- 全処理はローカル(ブラウザ内)で完結

**プライバシーポリシーURL**:
データを収集しないため必須ではないが、求められたら下記をGitHub等に
公開してURLを貼る:

```
本拡張機能「Board Image Downloader for Pinterest」は、個人情報・
閲覧履歴・認証情報を含む一切のユーザーデータを収集、保存、送信しません。
すべての処理はユーザーのブラウザ内で完結します。
Pinterestへのアクセスは、ユーザーが明示的にダウンロード操作を行った
ときに、ピン情報と画像の取得のためにのみ行われます。
```

## 提出時の注意 / リスク

1. **商標**: 名前を「Board Image Downloader for Pinterest」(for X形式)に
   している。先頭に「Pinterest」を置くと商標ポリシーで却下されやすい。
   アイコンもPinterestロゴ(P)は使っていないのでOK。
2. **非公式API**: Pinterestの内部APIを使っているため、Pinterest側から
   苦情(DMCA等)が入ると掲載停止される可能性はある。同種の拡張は
   多数掲載されているが、リスクとして認識しておく。
3. **審査期間**: host_permissionsを使う拡張は審査が長め(数日〜2週間)。
4. **アカウント**: 2段階認証の有効化が必須。

## 更新手順(次回以降)

1. manifest.jsonの`version`を上げる (例: 1.0.1)
2. `cd extension && zip -r ../dist/<name>-<ver>.zip . -x '.*'`
3. ダッシュボード → 対象アイテム → 「パッケージ」→ 新しいZIPをアップロード → 審査へ
