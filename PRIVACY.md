# プライバシーポリシー / Privacy Policy

**Board Image Downloader for Pinterest**

最終更新日 / Last updated: 2026-06-11

## 日本語

本拡張機能「Board Image Downloader for Pinterest」(以下「本拡張」)は、
ユーザーのプライバシーを尊重します。

### データの収集について

本拡張は、**いかなるユーザーデータも収集、保存、送信、販売しません。**

- 個人情報、閲覧履歴、認証情報、Cookieなどを収集しません
- 開発者や第三者のサーバーへデータを送信しません
- 解析ツール(アナリティクス)や広告を含みません

### 処理の内容

すべての処理はユーザーのブラウザ内で完結します。

- Pinterest(pinterest.com / pinterest.jp / i.pinimg.com)へのアクセスは、
  ユーザーがポップアップから明示的にダウンロード操作を行ったときに、
  開いているボードのピン情報と画像の取得のためにのみ行われます
- 取得した画像はユーザーのローカルのダウンロードフォルダにのみ保存されます

### 権限の使用目的

| 権限 | 目的 |
| --- | --- |
| `downloads` | 画像/ZIPをダウンロードフォルダに保存するため |
| `offscreen` | 画像をZIPにまとめる処理をブラウザ内で実行するため |
| `pinterest.com` / `pinterest.jp` へのアクセス | 開いているボードのピン一覧を取得するため |
| `i.pinimg.com` へのアクセス | 画像ファイル本体を取得するため |

## English

The browser extension "Board Image Downloader for Pinterest" (the
"Extension") respects your privacy.

### Data Collection

The Extension **does not collect, store, transmit, or sell any user data.**

- No personal information, browsing history, credentials, or cookies are collected
- No data is sent to the developer or any third-party server
- No analytics or advertising is included

### How It Works

All processing happens locally in your browser. The Extension accesses
Pinterest (pinterest.com / pinterest.jp / i.pinimg.com) only when you
explicitly start a download from the popup, solely to fetch the pin list
and image files of the board you are viewing. Downloaded images are saved
only to your local Downloads folder.
